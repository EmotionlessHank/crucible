# crucible

> 加密波段量化炼蛊机 —— 一个坩埚:炼脱过拟合杂质,只让验证存活的真货毕业拿真钱。

一台达尔文式的策略筛选机。它不「找一个好策略」,而是**持续生成候选 → 用严苛验证关卡淘汰过拟合垃圾 → 只让幸存者毕业拿真钱**。机器的价值在淘汰机制本身可信,不在某一个策略。

- **主战场**:大流动性 BTC(波段,持仓数小时~数天),后续可迁 IBKR。
- **资金结构**:核心-卫星杠铃。核心仓已 BTC 定投/分批;本机器是 ~1% 实验卫星仓,博高赔率。真实交付物 = 可信策略 + 不自欺的机器。
- **目标函数**:绝对收益(年化为正 + 控回撤);年化 100%+ 是「验证过的 edge × 杠杆」的期望产出,**不是准入门槛**。

## 核心设计

- **双层自迭代**:内循环(全自动,参数搜索)+ 外循环(人播种新 edge 假设,由交易日志失败归因驱动)。
- **fitness 抗过拟合**:机器朝「先存活,再风险调整最优(Calmar)」爬山,而非朝收益爬 —— 这是全自动机器不进化成骗子的根本保证。
- **存活式准入**:6 条硬门槛(walk-forward / DSR / PBO / 参数高原 / 净 edge / 纸面站得住),**无收益门槛**。
- **分阶段**:Phase 0 搭机器 → Phase 1 纸面自迭代(1 月时间盒)→ Phase 2 真钱(1% 卫星仓,全自动上线 + 硬护栏)。
